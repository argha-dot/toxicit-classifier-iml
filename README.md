
## Directory structure
 - `data` (should contain the csv files)
 - `src` (contains all the code and notebooks)
   - classifiers (contains `.pkl` files for saving the model)
   - `*.ipnyb`
 - `.gitignore`
 - `README.md`
 - `requirements.txt`

## How to run.
 - Do `pip install -r requirements.txt`
 - Create a folder named `data` in the root directory if it isn't there.
 - Download the dataset from [here](https://www.kaggle.com/c/jigsaw-toxic-comment-classification-challenge) and paste test.csv and train.csv in the `data` folder.
 - Run the code
